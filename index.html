  • git clone "путь" скачать с репозитория
  • git status показывает производились ли изменения
  • git add "файл" добивать файл в окружение ГИТа - на твоем ПК
  • git commit -am "added new file index.html" - зафиксировать и дать комментарий
  • git push - отправить файл на сервер (без указания файла)
  • git pull - забрать с сервера файлы
